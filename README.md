## 📌 About the Project

**Mana Raithu Market** is a full-stack agricultural marketplace platform
that helps farmers in Andhra Pradesh sell crops directly to hotels,
restaurants, supermarkets, event organizers, and wholesalers — without
any middlemen.

Farmers earn up to **68% more income** compared to traditional markets
by connecting directly with verified buyers across Andhra Pradesh.

## 🚀 Upcoming Features (Java + Database)

The next phase of this project will add a full backend using:

- **Java (Spring Boot)** — REST API for all platform features
- **MySQL Database** — Farmer profiles, crop listings, orders, payments
- **JWT Authentication** — Secure login for farmers and buyers
- **Aadhaar Verification API** — Real farmer identity verification
- **Razorpay Payment Gateway** — UPI and bank transfer support
- **SMS/WhatsApp Alerts** — Order and delivery notifications in Telugu
- **AI Price Engine** — Real-time market price prediction
- **Admin Dashboard** — Platform management and analytics
